## Glossary
hits:                   Total number of requests in log file.
median_duration_ms:     Median response time for puma process, in milliseconds
90p_duration_ms:        90th percentile of response time for puma process, in milliseconds
                        (90 % of requests are faster than this)
total_duration_ms:      total time spent processing requests, in milliseconds (Equivalent to hits * mean_duration_ms)
mean_memory_growth_mb:  Mean delta resident size (MB) between two consecutive requests by the same PID.
                        Note memory is reported as an integer.
                        Also note that because puma is multithreaded, delta resident size is expected to 
                        bleed onto other concurrent requests.
total_memory_growth_mb: hits * mean_memory_growth_mb
